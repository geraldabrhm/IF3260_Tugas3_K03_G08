Model Controller

Transformasi dapat dilakukan dengan memilih model terlebih dahulu dari dropdown. Transformasi yang dapat dilakukan adalah rotasi, translasi, dan scale terhadap sumbu X, Y, maupun Z.

a. Rotasi

Parameter rotasi dapat diatur dengan slider berlabel Rotasi. Rotasi dilakukan dengan titik tengah model sebagai pusat rotasi.

b. Translasi

Parameter translasi dapat diatur dengan slider berlabel Translasi.

c. Scale

Parameter scale dapat diatur dengan slider berlabel Scale.

Global Controller

Menu pengaturan di Global Controller digunakan untuk mengatur konfigurasi view secara keseluruhan meliputi jenis proyeksi, camera, dan lighting.

Tombol Reset View digunakan untuk mengembalikan seluruh konfigurasi di Global Controller ke default value. Tombol Clear Canvas digunakan untuk menghapus seluruh model di dalam canvas.

a. Proyeksi

Jenis proyeksi dapat diatur dengan memilih dari dropdown berlabel Select projection type. Jenis proyeksi yang tersedia adalah Perspective (default), Oblique, dan Orthographic.

b. Camera

Konfigurasi camera dapat diatur dengan slider berlabel Camera Angle dan Camera Radius. Camera Angle digunakan untuk mengatur posisi sudut camera sehingga dapat mengitari model. Camera Radius digunakan untuk mengubah jarak kamera view.

c. Lighting / Shading

Konfigurasi lighting dapat diatur dengan slider berlabel Light Position yang berfungsi untuk mengatur arah datang cahaya. Lighting dapat dimatikan atau dinyalakan dengan checkbox berlabel Light Shader. Cahaya berasal dari jarak infinity yang mensimulasikan cahaya dari matahari.

Export

Menu export digunakan untuk menyimpan model ke dalam bentuk file json. Tombol Export menyimpan seluruh model. Sedangkan tombol Export Selected hanya menyimpan model yang dipilih pada dropdown seleksi model

Import

Menu import digunakan untuk load model dari file dengan format json. Model yang telah disiapkan tersedia di folder test repo proyek ini.
